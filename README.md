# Higher Timeframe Candle Overlay

- A lightweight and efficient cTrader indicator that overlays higher timeframe candles directly on your current chart.




## Overview

- This indicator draws complete higher timeframe candlesticks (body + wick) on your lower timeframe chart, helping you maintain multi-timeframe awareness without switching screens.

## Features

- Dynamic higher timeframe candles (H1, H4, Daily, Weekly, etc.)
- Customizable bullish & bearish colors
- Adjustable wick thickness
- Automatic update on scroll, zoom, and new bars
- Efficient rendering only draws visible candles
- Index offset option for fine alignment adjustments




## Example Use Cases

- Trading M15 while seeing H1 or H4 structure
- Monitoring Daily bias on lower timeframes
- Quick higher-timeframe context during scalping or day trading
